3️⃣ Строим LPS пошагово
	•	i = 0 → lps[0] = 0
	•	i = 1 → s[1]=‘b’, s[0]=‘a’, нет совпадения → lps[1]=0
	•	i = 2 → s[2]=‘a’, s[0]=‘a’, совпало → lps[2]=1
	•	i = 3 → s[3]=‘b’, s[1]=‘b’, совпало → lps[3]=2
	•	i = 4 → s[4]=‘c’, s[2]=‘a’, не совпало → fallback j=lps[1]=0 → нет → lps[4]=0
	•	i = 5 → s[5]=‘a’, s[0]=‘a’, совпало → lps[5]=1
	•	i = 6 → s[6]=‘b’, s[1]=‘b’, совпало → lps[6]=2
	•	i = 7 → s[7]=‘a’, s[2]=‘a’, совпало → lps[7]=3
	•	i = 8 → s[8]=‘b’, s[3]=‘b’, совпало → lps[8]=4

i:   0 1 2 3 4 5 6 7 8
s:   a b a b c a b a b
lps: 0 0 1 2 0 1 2 3 4


3️⃣ Как найти самый большой суффикс-префикс для всей строки
	•	Берём последний элемент LPS, т.е. lps.back()
	•	Это длина наибольшего префикса, который является суффиксом всей строки

В нашем примере:
lps.back() = lps[8] = 4
Ответ: 4

4️⃣ Как узнать сам сам суффикс-префикс
	•	Если длина = len = lps.back() = 4
	•	Префикс = s.substr(0, len) → "abab"
	•	Суффикс = s.substr(s.size() - len, len) → "abab"

В нашем примере:
s.substr(0, 4) = "abab"
s.substr(9 - 4, 4) = s.substr(5,4) = "abab"
